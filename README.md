## Full Stack To-Do List App

This is a full stack To-Do List application.

## Technologies Used
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MongoDB

## Features
- Add tasks
- Delete tasks
- Tasks stored in MongoDB
- Backend API using Express

## Project Structure
```text
To_Do_List
│
├── backend
│   ├── server.js
│   ├── routes
│   ├── model
│
└── frontend
    ├── index.html
    ├── script.js
    ├── style.css
```

## How to Run

1. Start MongoDB
-mongod

2. Start backend
-cd backend
-node server.js

3. Open frontend
-frontend/index.html
